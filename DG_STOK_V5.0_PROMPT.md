# DG STOK V5.0 - Profesyonel XML Entegrasyon ve Pazaryeri ERP Sistemi

Profesyonel seviyede çalışan, modern, hızlı ve tamamen otomatik çalışan bir XML Entegrasyon + ERP + Pazaryeri Yönetim Sistemi geliştir.

Amaç; XML tedarikçilerinden ürünleri çekmek, tüm verileri düzenlemek, eşleştirmek, optimize etmek ve Trendyol, Hepsiburada, N11, Amazon, Pazarama, ÇiçekSepeti, İdefix, PttAVM, WooCommerce, Shopify gibi tüm pazaryerlerine tek panelden yönetmektir.

Tasarım Apple, Linear, Vercel ve Stripe kalitesinde olmalı.

Kullanıcı arayüzü sade, premium, hızlı ve profesyonel görünmelidir.

Dark Mode varsayılan olsun.

Responsive yapı kullanılmalı.

Tüm sayfalarda loading, skeleton, toast notification, modal, tooltip ve animasyonlar bulunmalıdır.

# XML Kaynakları

Bu bölüm ekran görüntüsündeki yapıya benzeyecek fakat çok daha gelişmiş olacak.

Tablo sütunları:
Kaynak Adı, Firma, XML URL, XML Tipi, Durum, Son Senkronizasyon, Toplam Ürün, Yeni Ürün, Güncellenen, Silinen, Hata, İşlemler

Her satırda; Senkronize Et, Log, Düzenle, Sil, Pasif Yap, XML Test butonları olacak.

Yeni XML eklerken; Firma, XML Linki, Kullanıcı, Şifre, XML Tipi, Para Birimi, Vergi, Stok Güncelle, Fiyat Güncelle, Resim Güncelle, Aktif/Pasif, Otomatik Çalışma Süresi, Cron Ayarı bulunacak.

XML eklendiğinde sistem; ürün sayısını, kategori sayısını, marka sayısını, varyant sayısını otomatik analiz edecek.
